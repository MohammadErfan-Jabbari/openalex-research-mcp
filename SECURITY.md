# Security Policy

## Supported Versions

Security fixes are provided on `master` and the latest published release line.

## Reporting a Vulnerability

Report security issues privately via GitHub Security Advisories:

- https://github.com/MohammadErfan-Jabbari/openalex-research-mcp/security/advisories/new

Please include:

- Affected version or commit SHA
- Reproduction steps
- Impact assessment
- Suggested fix (if available)

We will acknowledge reports quickly and coordinate a fix before public disclosure.
